## 🏗️ Proyecto Actual: Grupo Supply E-commerce
Actualmente, lideramos el desarrollo del ecosistema digital para **Grupo Supply**. Este proyecto representa un hito en la automatización de ventas para el sector de suministros, gestionando un catálogo masivo de más de 150 productos con inteligencia de negocios integrada.

### **Puntos clave del ecosistema:**
* **Inteligencia de Precios:** Motor de descuentos dinámicos por volumen y bonificaciones por método de pago.
* **Arquitectura Escalable:** Estructura modular basada en NestJS y Next.js para garantizar alta disponibilidad y velocidad.
* **Diseño UI/UX de Autor:** Interfaces diseñadas desde cero para maximizar la conversión y la facilidad de uso móvil.
* **Automatización Operativa:** Generación automática de notas de entrega y sincronización directa con WhatsApp Business.

## 🛠️ Nuestro Stack Tecnológico
Seleccionamos herramientas de última generación para garantizar que cada producto sea rápido, seguro y fácil de mantener:

* **Frontend:** `Next.js 14 (App Router)`, `TypeScript`, `Tailwind CSS`.
* **Backend:** `NestJS`, `Node.js`, `PostgreSQL`.
* **Infraestructura:** `Cloudinary/S3` para Media, `Docker`, `CI/CD via GitHub Actions`.
* **Diseño:** Principios de `Screaming Architecture` y `Modular Design`.
